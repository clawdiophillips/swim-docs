# Keyboard Shortcuts

Keyboard shortcuts dramatically speed up your annotation workflow. Learn these and you'll rarely need to touch the mouse when marking up a race.

---

## Video Playback

| Shortcut | Action |
|---|---|
| `Space` | Play / Pause |
| `←` | Step back one frame |
| `→` | Step forward one frame |
| `Shift + ←` | Step back 10 frames |
| `Shift + →` | Step forward 10 frames |
| `J` | Rewind (hold for continuous rewind) |
| `K` | Pause |
| `L` | Fast forward (hold for continuous) |
| `0` – `9` | Jump to 0%–90% of video |
| `Home` | Jump to start of video |
| `End` | Jump to end of video |

---

## Playback Speed

| Shortcut | Speed |
|---|---|
| `,` (comma) | Decrease playback speed |
| `.` (period) | Increase playback speed |

Available speeds: 0.25×, 0.5×, 0.75×, 1×, 1.5×, 2×

---

## Adding Markers

| Shortcut | Marker type |
|---|---|
| `S` | Start |
| `T` | Turn |
| `B` | Breakout |
| `F` | Finish |
| `U` | 15m / Underwater |
| `R` | Reaction |
| `M` | Custom |

All shortcuts place the marker at the **current playhead position**.

---

## Editing Markers

| Shortcut | Action |
|---|---|
| `Enter` | Update selected marker's time to current playhead |
| `Delete` / `Backspace` | Delete selected marker |
| `Escape` | Deselect marker |
| Click marker + `←` / `→` | Nudge selected marker one frame |

---

## Video Trim Mode

| Shortcut | Action |
|---|---|
| `I` | Set trim in-point at current position |
| `O` | Set trim out-point at current position |
| `P` | Preview the trimmed clip |

---

## Comparison Mode

| Shortcut | Action |
|---|---|
| `Space` | Play / Pause both videos |
| `←` / `→` | Step both videos one frame |
| `Shift + ←` / `→` | Step both videos 10 frames |

---

## General

| Shortcut | Action |
|---|---|
| `?` | Open keyboard shortcut reference overlay |
| `Cmd + Z` / `Ctrl + Z` | Undo last marker action |
| `Cmd + Shift + Z` / `Ctrl + Shift + Z` | Redo |
| `Escape` | Close any open panel or dialog |

---

> **Mac users:** Replace `Ctrl` with `Cmd` for undo/redo. All other shortcuts are identical across Mac, Windows, and Linux.
